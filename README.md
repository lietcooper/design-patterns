# Design Patterns
This project implements 23 design patterns using C++ with one example for each pattern.

## Content
- Creational Design Patterns
  - Singleton Pattern
  - Factory Method Pattern
  - Abstract Factory Pattern
  - Builder Pattern
  - Prototype Patter
- Behavioral Design Patterns
  - Adapter Pattern
  - Decorator Pattern
  - Proxy Pattern
  - Facade Pattern
  - Composite Pattern
  - Bridge Pattern
  - Flyweight Pattern
- Structural Design Patterns
  - Observer Pattern
  - Strategy Pattern
  - Command Pattern
  - Chain of Responsibility Pattern
  - Template Method Pattern
  - Iterator Pattern
  - State Pattern
  - Mediator Pattern
  - Memento Pattern
  - Visitor Pattern

# Compile and Run

To compile a cpp file, for example:
```bash
g++ singleton -o singleton.exe
```

To execute:
```bash
./singleton.exe
```
